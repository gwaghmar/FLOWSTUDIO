# Cloud provider icons

Official AWS / GCP / Azure architecture icons, sourced from
[mingrammer/diagrams](https://github.com/mingrammer/diagrams) (MIT licensed
redistribution of each vendor's official icon set for use in diagram tooling —
the same intended use case as this app). Unmodified, per each vendor's icon
usage guidelines.

`manifest.json` maps `provider -> service-token -> icon path`, consumed by
`apps/web/lib/diagrams/provider-icons.ts`.
